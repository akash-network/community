## Resource Reclamation & Graceful Shutdowns

### Resource Reclamation Feature

* Artur introduced a new marketplace feature called Resource Reclamation
* Feature designed to improve interactions between providers and tenants
* Addresses situations where providers need to temporarily reclaim hardware resources

### Graceful Shutdown Mechanism

* New system introduces a provider-initiated reclamation process
* Providers will be required to notify tenants before reclaiming resources
* Notification occurs through an on-chain transaction

### Planned Workflow

* Provider submits a reclamation request
* Provider specifies a predefined notification window
* Example:
  * Provider may notify user 24 hours before resource reclamation
* Provider cannot immediately terminate a lease
* Lease termination only permitted after notification period expires

### Expected Benefits

* Improved tenant experience
* Reduced unexpected service interruptions
* Greater transparency between providers and users
* Better resource management flexibility for providers

## Consensus Vulnerability Discussion

### Security Disclosure Raised by Austin Hardnett

* Austin Hardnett raised awareness of a recently disclosed CometBFT consensus vulnerability
* Vulnerability reportedly prevents nodes from properly re-syncing after going offline
* Disclosure had been made publicly rather than through a private disclosure process

### Engineering Team Response

* Artur Troian was initially unaware of the disclosure
* Requested links and supporting documentation for investigation
* Austin shared:
  * Public disclosure thread
  * Security advisory references
  * Version information

### Version Review

* Vulnerability impacts:
  * CometBFT versions greater than or equal to v0.38.16
  * Versions less than or equal to v1.0.0
* Artur confirmed Akash currently operates on:
  * CometBFT v0.38.21
* Initial assessment indicated Akash is not affected by the disclosed issue

### Follow-Up Actions

* Artur committed to contacting Cosmos ecosystem teams for clarification
* Tyler Wright thanked Austin for surfacing the issue
* Community members encouraged to continue reporting ecosystem security developments between meetings
