* The goal is to provide clearer onboarding for new contributors joining the Akash ecosystem
* The community will be informed once these updates are finalized and deployed

### BME Incentivized Testnet Launch Announcement

* Tyler Wright shared an update on the upcoming Burn-Mint Equilibrium incentivized testnet
* Internal testing has been conducted by the core engineering team and selected members of the Akash Insiders and Akash Club programs
* Several bugs have already been identified and resolved during internal testing phases
* The incentivized testnet is planned to launch on Tuesday, February 17, 2026
* The testing program is expected to run for approximately two weeks
* Participants will be able to submit testing results through an official submission form

* Piyush confirmed that the banner can be implemented on the website starting Tuesday
* Michelle will provide the final banner messaging and call-to-action text

### Student Ambassador Page Improvements

* Tyler introduced a GitHub issue requesting updates to the Student Ambassador page
* The changes aim to improve presentation and highlight participating universities
* A carousel section will be added to display student activity images and program highlights
* The carousel will likely appear between the application section and the “Student Voices” section
* University logos will be displayed at the bottom of the page
* Logos should be presented in full color rather than monochrome versions
* Additional images will be collected to expand the existing gallery

### Implementation Plan for Student Ambassador Page

* The carousel design will follow a format similar to other carousel elements already used on the website
* The apply section will remain the primary call-to-action for the page
* The image gallery will be expanded to better represent the ambassador program
* Piyush will coordinate with Tyler to obtain additional images and assets

### Newsletter Subscription Infrastructure Update

* The updated newsletter system will support both marketing newsletters and community newsletters
* The signup URL will change once the new tool is implemented
