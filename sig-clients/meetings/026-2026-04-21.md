* Tyler Wright handed discussion over to Joshua Meyer from the core product and design team

### Onboarding Flow Redesign

* Product and design teams actively redesigning the Akash onboarding experience
* Team using an analytics-driven approach to identify major user drop-off points
* Existing onboarding flow being reworked to:
  * Reduce friction
  * Simplify onboarding
  * Improve intuitiveness
  * Improve account security
  * Reduce free trial abuse
* Design and product teams currently finalizing:
  * User flows
  * Copywriting
  * Interaction details
* Development expected to begin shortly after internal review completion
* Team expressed strong confidence in the new onboarding direction

## Reverse Auction & Provider Selection Improvements

* Joshua Meyer shared updates regarding redesign of the deployment and provider selection experience
* Current reverse auction workflow identified as an area requiring usability improvements
* Product and engineering teams collaborating on technical implementation plans

### Planned Improvements

* Users will be able to preview potential providers while configuring deployment requirements
* Goal is to create a more familiar marketplace-style experience
* Improvements intended to:
  * Help users make more informed deployment decisions
  * Improve provider visibility
  * Reduce failed deployment attempts
  * Reduce confusion during provider selection
* Current issue identified where users:
  * Build SDLs
  * Submit deployments
  * Reach provider selection stage
  * Receive few or no bids
* Planned redesign aims to significantly reduce this friction

### Deployment Experience Optimization

* Team also focused on reducing the amount of time between:
  * Deployment creation
  * Provider selection
  * Deployment execution
* Additional work still required around:
  * Technical implementation
  * UX refinement
  * Marketplace interaction flows
* Development expected to begin once revisions are finalized

## Home Node Beta Program Updates

* Tyler Wright shared updates regarding [Akash Home Node](https://homenode.akash.network/)
* Home Node discussed as a client-focused initiative designed to simplify provider onboarding

### Home Node Early Access Beta

* Active beta currently available for users with:
  * NVIDIA RTX 4090 GPUs
  * NVIDIA RTX 5090 GPUs
* Home Node designed to allow individuals to:
  * Join provider clusters
  * Contribute compute resources
  * Earn rewards through Akash Network
* Goal is to abstract away complexity involved in becoming an Akash provider

### Accessibility & Simplified Provider Onboarding

* Home Node intended to simplify the provider onboarding process for less technical users
* Traditional provider setup recognized as:
  * Structured
  * Well-documented
  * Potentially intimidating for newer participants
* Home Node provides a simplified ISO-based onboarding experience
* Additional GPU support expected in future releases

### Community Participation

  * Join the beta
  * Test Home Node
  * Provide feedback
  * Participate in ecosystem discussions
* Tyler directed interested users to:
