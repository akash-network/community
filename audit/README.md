# Provider audit

Official documentation for getting your Akash provider audited:

**[Provider Audit — akash.network/docs](https://akash.network/docs/providers/operations/provider-audit/)**

That guide covers attribute requirements, hardware minimums, deploying the benchmark, and opening a **[Provider Audit]** GitHub issue after all tests pass.

## Files in this repo

- **[standalone-audit.sdl.yaml](./standalone-audit.sdl.yaml)** — SDL to deploy for the audit benchmark (set `AUDIT_PROVIDER_ID` to your provider owner address)

## Submit an audit request

When benchmarks finish and **all tests pass** in the audit UI, open a **[Provider Audit]** issue: [github.com/akash-network/community/issues](https://github.com/akash-network/community/issues/new)
